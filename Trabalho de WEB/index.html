Regras

🎯 Objetivo do jogo
Descobrir a palavra secreta de 5 letras em no máximo 6 tentativas.

🕹️ Como jogar
Digite uma palavra válida de 5 letras e envie (geralmente pressionando Enter).
Após cada tentativa, o jogo fornece dicas de cores para cada letra:
  • 🟩 Verde (correto) – A letra está na posição certa.
  • 🟨 Amarelo (parcial) – A letra existe na palavra, mas está na posição errada.
  • ⬛ Cinza (errado) – A letra não está presente na palavra.
Use o feedback de cores para ajustar suas próximas tentativas.

⏱️ Regras adicionais
  • As palavras tentadas devem existir no dicionário do jogo.
  • Você tem 6 tentativas para acertar.
  • A mesma letra pode aparecer mais de uma vez.
